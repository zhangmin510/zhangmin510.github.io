SDK

  1. Guacamole是一个HTML5 Web应用程序，使用像VNC or RDP这样的远程桌面技术来访问远程电脑的桌面。它可作为集中服务器的一个隧道和代理，这样可以通过Web浏览器访问多个桌面。无需插件，客户端只需要一个支持HTML5和AJAX的Web浏览器。
  2. libvnc VNC的C语言版本客户端和服务端类库。
  3. minimega中的vnc功能
  4. TigerVNC多平台高性能的VNC客户端和服务端类库。
  5. RealVNC Java SDK收费版本
  6. TightVNC远程控制软件，提供Java，C，C#的SDK。

VNC会话录制

  1. vnc2flv。将vnc会话录制为flv格式，可以通过ffmpeg转化为其他视频格式。
