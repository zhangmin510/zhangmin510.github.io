My journey from Python to Go

这篇文章作者分享了自己作为一名Python开发者转向Go开发者中的历程。介绍了Python和Go两种语言最关键的不同之处以及相关的学习资料。

Python和Go最关键的不同：

  1. 项目结构。 Go有明确的项目结构。利用模块化和大型项目的维护。
  2. 静态类型语言。
  3. 内建的并发机制goroutine和channel。
  4. JSON序列化和反序列化。
  5. 清洁代码。Go自带的gofmt以及_。
  6. 寻找适合的类库和框架。
